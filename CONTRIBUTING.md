# Contributing to Project Aurora

Thanks for your interest in contributing! We welcome pull requests, bug reports, and feature suggestions. Here are some quick guidelines:

- Fork the repository and create a feature branch for your change.
- Follow the coding conventions used in the project (see `.editorconfig`).
- Write clear, small commits with descriptive messages.
- Add tests for new features or bug fixes when applicable.
- Run `dotnet build` and `dotnet test` before submitting a PR.
